# تطبيق منظم المهام اليومي

تطبيق ويب تقدمي (PWA) لتنظيم وإدارة مهامك اليومية بشكل سهل وفعال.

## المميزات

✅ **إضافة مهام جديدة** - أضف مهامك مع تحديد الأولوية (عالية، متوسطة، منخفضة)
✅ **تعديل المهام** - عدّل على مهامك الموجودة بسهولة
✅ **حذف المهام** - احذف المهام المكتملة أو غير المرغوب فيها
✅ **تحديد الأولويات** - لوّن المهام حسب أولويتها (أحمر للعالية، أصفر للمتوسطة، أخضر للمنخفضة)
✅ **فلترة المهام** - اعرض جميع المهام، المكتملة فقط، أو قيد الانتظار فقط
✅ **إحصائيات** - تابع عدد المهام الكلي، المكتملة، والقيد الانتظار
✅ **حفظ تلقائي** - تُحفظ مهامك تلقائياً في المتصفح
✅ **يعمل بدون إنترنت** - تطبيق PWA يعمل حتى بدون اتصال بالإنترنت
✅ **تصميم متجاوب** - يعمل على جميع أحجام الشاشات (هاتف، تابلت، كمبيوتر)
✅ **واجهة عربية** - واجهة مستخدم كاملة باللغة العربية

## كيفية الاستخدام

### على الهاتف المحمول:
1. افتح التطبيق في متصفح هاتفك
2. اضغط على زر "+" العائم لإضافة مهمة جديدة
3. اكتب نص المهمة واختر الأولوية
4. اضغط "إضافة" أو Enter

### على الكمبيوتر:
1. افتح التطبيق في المتصفح
2. اكتب المهمة في حقل الإدخال
3. اختر الأولوية من القائمة المنسدلة
4. اضغط زر "إضافة"

### إدارة المهام:
- ✅ **إكمال مهمة**: اضغط على مربع الاختبار بجانب المهمة
- ✏️ **تعديل مهمة**: اضغط على أيقونة القلم
- 🗑️ **حذف مهمة**: اضغط على أيقونة سلة المهملات

### الفلترة:
- **الكل**: عرض جميع المهام
- **قيد الانتظار**: عرض المهام غير المكتملة فقط
- **مكتملة**: عرض المهام المكتملة فقط

## التثبيت كتطبيق (PWA)

### على Android (Chrome):
1. افتح التطبيق في Chrome
2. اضغط على القائمة (ثلاث نقاط)
3. اختر "تثبيت التطبيق" أو "إضافة إلى الشاشة الرئيسية"

### على iOS (Safari):
1. افتح التطبيق في Safari
2. اضغط على زر المشاركة
3. اختر "إضافة إلى الشاشة الرئيسية"

### على الكمبيوتر:
1. افتح التطبيق في Chrome أو Edge
2. اضغط على أيقونة التثبيت في شريط العنوان

## التقنيات المستخدمة

- HTML5
- CSS3 (مع Flexbox و Grid)
- JavaScript (ES6+)
- LocalStorage API
- Service Worker (للدعم دون اتصال)
- Web App Manifest


## التشغيل المحلي

يمكنك تشغيل التطبيق محلياً باستخدام أي خادم ويب بسيط:

### باستخدام Python:
```bash
cd todo_app
python -m http.server 8000
```

ثم افتح المتصفح على: `http://localhost:8000`

### باستخدام Node.js:
```bash
cd todo_app
npx serve
```
